# TabBar 图标说明

请在此目录下放置以下图标文件（PNG 格式，尺寸建议 81x81 px）：

## 新闻图标
- `news.png` - 未选中状态（灰色）
- `news-active.png` - 选中状态（蓝色 #1890ff）

## 地图图标
- `map.png` - 未选中状态（灰色）
- `map-active.png` - 选中状态（蓝色 #1890ff）

## 图标设计建议

### 新闻图标
可以使用报纸、文档或消息相关的图标

### 地图图标
可以使用定位针或地图相关的图标

你可以使用以下工具创建图标：
1. 在线工具：iconfont.cn、fontawesome.com
2. 设计工具：Figma、Sketch、Adobe XD
3. AI 生成：使用 AI 工具生成简单的图标

注意：微信小程序的 tabBar 目前不支持 SVG 格式，必须使用 PNG 格式。
